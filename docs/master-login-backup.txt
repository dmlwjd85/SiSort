침묵의 가나다 — 마스터 계정 정리·재설정 (SiSort)
===============================================

[1] 기존 마스터 삭제 (Firebase 콘솔)
- Authentication → Users → master_*@sisort.local 사용자 선택 → 삭제
- 앱 코드로 계정 삭제는 불가(관리자 SDK 필요). 콘솔에서만 삭제합니다.

[2] 앱에서 깔끔히 다시 만들기 (권장)
- 배포된 사이트 → 로그인 화면 → «마스터» → «최초 설정»
- 초기 번호(6~7자리) 두 번 입력 → «마스터 가입하고 들어가기»
- 이메일은 자동으로 master_같은숫자@sisort.local 이 되고, 비밀번호도 그 숫자와 동일합니다.

[3] (선택) 배포 빌드에만 최초 설정 키를 쓰는 경우
- 로컬 .env 또는 GitHub Secrets: VITE_MASTER_SETUP_SECRET
- 값이 비어 있으면 키 입력 없이 최초 설정 가능(누가 먼저 번호를 잡느냐에 따라 주의)
- 값이 있으면 «최초 설정» 시 같은 키를 입력해야 가입됩니다.

[4] 이미 가입한 뒤
- «마스터» → «로그인»에서 번호만 입력

[권한]
- master_*@sisort.local 로 로그인 시 앱이 마스터 권한을 부여합니다.
